streamlist 
streamlit is used to build web UI for your pythin app
creates buttons,textboxes,chain inputs

langchain
langchain is a framework for our application
uses llms,documents,embedding,prompts,chains

langchain-community
we are using community for extra integration maintained by this library
for example,ollama connectors,document loaders,hugging face embeddings

langchain-core
core building blocks of langchain
runnables,output parsers,prompt template

ollama
runs local LLMs on your PC
